NAME
    share secure logs from the command line

SYNOPSIS
    $ curl <%- BASE_URL %> --data-binary 'hello world'
    you can also paste or drag files to this page

DESCRIPTION
    works better with text
    limited to 20MiB
    everything are encrypted
    decryption only on browser
    no data or logs are collected

EXAMPLES
    $ curl <%- BASE_URL %> --data-binary @debug.log
    $ git diff --staged | curl <%- BASE_URL %> --data-binary @-
    add to your .profile
    $ logsmoe() { curl <%- BASE_URL %> --data-binary "@${1:--}"; }
    use with file or stdin
    $ logsmoe debug.log
    $ git diff --staged | logsmoe
